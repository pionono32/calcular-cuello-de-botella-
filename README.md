# BottleneckCalc - Calculadora de Bottleneck PC

Web completa orientada a SEO sobre análisis de cuello de botella entre CPU y GPU.

## Estructura

- **24 páginas HTML** con SEO completo
- **70+ CPUs** y **60+ GPUs** en base de datos
- Calculadora avanzada con análisis por resolución y tipo de uso
- Guías prácticas y blog con contenido original

## Cómo subir a GitHub Pages

### Paso 1: Crear repo en GitHub
1. Andá a [github.com](https://github.com) y creá un nuevo repositorio
2. Nombralo `fps-check` (o el nombre que prefieras)
3. Dejalo público

### Paso 2: Subir archivos
1. Dentro del repo, clickeá "Add file" → "Upload files"
2. Arrastrá **toda la carpeta** `fps-check` (todos los archivos y subcarpetas)
3. Confirmá el upload

### Paso 3: Activar GitHub Pages
1. Andá a "Settings" del repo
2. En el menú lateral, clickeá "Pages"
3. En "Source", seleccioná "Deploy from a branch"
4. Elegí la branch `main` y carpeta `/ (root)`
5. Guardá y esperá 1-2 minutos

### Paso 4: Verificar
- Tu sitio va a estar en: `https://TU-USUARIO.github.io/fps-check/`
- Reemplazá `TU-USUARIO` con tu username de GitHub

### Paso 5: Actualizar URLs (importante)
Buscá y reemplazá en todos los archivos HTML:
- `USERNAME.github.io/fps-check` → tu URL real de GitHub Pages
